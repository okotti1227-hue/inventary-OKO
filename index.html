🔧 Настройка Firebase

Шаг 1: Создайте проект

1. Перейдите на console.firebase.google.com
2. Нажмите "Create project"
3. Введите название (например "Inventory")
4. Отключите Google Analytics (для простоты)
5. Нажмите "Create project"

Шаг 2: Включите Realtime Database

1. В меню слева выберите "Realtime Database"
2. Нажмите "Create Database"
3. Выберите регион (europe-west1 для России)
4. Нажмите "Next"
5. Выберите "Start in test mode" (пока)
6. Нажмите "Enable"

Шаг 3: Получите конфигурацию

1. Нажмите на значок шестеренки ⚙️ рядом с "Project Overview"
2. Выберите "Project settings"
3. В разделе "Your apps" нажмите "" (Web)
4. Зарегистрируйте приложение (nickname: inventory)
5. Скопируйте блок кода с firebaseConfig
6. Вставьте в этот файл вместо текущего конфига

✅ Готово!

После вставки конфигурации обновите страницу. Данные будут синхронизироваться между всеми устройствами автоматически!
